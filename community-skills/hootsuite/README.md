# Hootsuite

Integrate with Hootsuite to list, schedule, list and more.

## Tools

- **List Profiles** (`hootsuite_list_profiles`) — List social media profiles connected to your Hootsuite account. Returns service types, usernames, and profile IDs.
- **Schedule Message** (`hootsuite_schedule_message`) — Schedule a social media message via Hootsuite. Posts to one or more social profiles at a specified time.
- **List Messages** (`hootsuite_list_messages`) — List scheduled messages from Hootsuite. Optionally filter by state or date range.
- **Get Analytics** (`hootsuite_get_analytics`) — Get social media analytics from Hootsuite for a specific profile. Returns engagement metrics like followers, posts, and interactions.
- **List Streams** (`hootsuite_list_streams`) — List content streams configured in Hootsuite. Streams are feeds of social content (mentions, timelines, searches, etc.).
