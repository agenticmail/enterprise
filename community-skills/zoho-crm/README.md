# Zoho CRM

Integrate with Zoho CRM to list, get, create and more.

## Tools

- **List Records** (`zoho_list_records`) — List records from a Zoho CRM module (Leads, Contacts, Accounts, Deals, etc.).
- **Get Record** (`zoho_get_record`) — Get a single record from a Zoho CRM module by its ID.
- **Create Record** (`zoho_create_record`) — Create a new record in a Zoho CRM module. Provide the module name and field values.
- **Update Record** (`zoho_update_record`) — Update an existing record in a Zoho CRM module. Provide the module, record ID, and fields to update.
- **Search Records** (`zoho_search_records`) — Search records in a Zoho CRM module using criteria, email, phone, or word search.
