# Drift Conversational Marketing

Integrate with Drift Conversational Marketing to list, send, list and more.

## Tools

- **List Conversations** (`drift_list_conversations`) — List conversations from Drift. Returns conversation IDs, statuses, and contact information.
- **Send Message** (`drift_send_message`) — Send a message in a Drift conversation. Can send as a user (agent) or post a chat message.
- **List Contacts** (`drift_list_contacts`) — List contacts from Drift. Returns contact names, emails, and companies.
- **Get Conversation** (`drift_get_conversation`) — Get detailed information about a specific Drift conversation, including its messages.
- **Create Contact** (`drift_create_contact`) — Create a new contact in Drift. Provide at least an email address.
