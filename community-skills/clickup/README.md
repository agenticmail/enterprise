# ClickUp

Integrate with ClickUp to list, create, update and more.

## Tools

- **List Tasks** (`clickup_list_tasks`) — List tasks in a ClickUp list. Filter by status, assignees, and due dates.
- **Create Task** (`clickup_create_task`) — Create a new task in a ClickUp list. Specify name, description, assignees, priority, and due date.
- **Update Task** (`clickup_update_task`) — Update an existing ClickUp task. Change name, status, priority, description, assignees, or due date.
- **List Spaces** (`clickup_list_spaces`) — List all spaces in a ClickUp team/workspace. Returns space names, IDs, and feature settings.
- **List Lists** (`clickup_list_lists`) — List all lists within a ClickUp folder or space. Returns list names, IDs, and task counts.
