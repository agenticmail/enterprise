# LiveChat

Integrate with LiveChat to list, send, list and more.

## Tools

- **List Chats** (`livechat_list_chats`) — List active and recent chats from LiveChat. Returns chat summaries with visitor names and last messages.
- **Send Event** (`livechat_send_event`) — Send a message event to an active LiveChat conversation. The agent must be assigned to the chat.
- **List Agents** (`livechat_list_agents`) — List agents in the LiveChat account. Returns agent names, emails, and current availability status.
- **Get Chat** (`livechat_get_chat`) — Get full details of a specific LiveChat conversation including all messages and participants.
