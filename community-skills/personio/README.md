# Personio

Integrate with Personio to list, get, list and more.

## Tools

- **List Employees** (`personio_list_employees`) — List employees from Personio. Returns names, emails, departments, and positions.
- **Get Employee** (`personio_get_employee`) — Get detailed information about a specific Personio employee by their ID.
- **List Absences** (`personio_list_absences`) — List absence (time-off) records from Personio. Filter by employee, date range, or type.
- **Create Absence** (`personio_create_absence`) — Create an absence (time-off) request in Personio for a specific employee.
- **List Attendances** (`personio_list_attendances`) — List attendance records from Personio. Filter by employee and date range to see clock-in/clock-out data.
