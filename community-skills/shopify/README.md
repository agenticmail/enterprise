# Shopify

Integrate with Shopify to list, create, list and more.

## Tools

- **List Products** (`shopify_list_products`) — List products in the Shopify store. Returns product titles, statuses, vendors, and variant counts.
- **Create Product** (`shopify_create_product`) — Create a new product in the Shopify store. Provide a title and optional body HTML, vendor, product type, and tags.
- **List Orders** (`shopify_list_orders`) — List orders from the Shopify store. Optionally filter by status, fulfillment status, or financial status.
- **Get Order** (`shopify_get_order`) — Get details of a single Shopify order by its ID. Returns full order information including line items.
- **List Customers** (`shopify_list_customers`) — List customers in the Shopify store. Optionally search by query string.
