# Teamwork

Integrate with Teamwork to list, list, create and more.

## Tools

- **List Projects** (`teamwork_list_projects`) — List all projects in the Teamwork account. Returns project names, statuses, and company info.
- **List Tasks** (`teamwork_list_tasks`) — List tasks in a Teamwork project or tasklist. Returns task names, assignees, and due dates.
- **Create Task** (`teamwork_create_task`) — Create a new task in a Teamwork tasklist. Specify name, description, assignees, and due date.
- **List Milestones** (`teamwork_list_milestones`) — List milestones in a Teamwork project. Returns milestone names, deadlines, and completion status.
- **Log Time** (`teamwork_log_time`) — Log time against a Teamwork task. Specify hours, minutes, date, and an optional description.
