# Lattice Performance

Integrate with Lattice Performance to list, get, list and more.

## Tools

- **List Users** (`lattice_list_users`) — List users from Lattice. Returns names, emails, titles, and department information.
- **Get User** (`lattice_get_user`) — Get detailed information about a specific Lattice user by their ID.
- **List Goals** (`lattice_list_goals`) — List goals from Lattice. Optionally filter by owner, status, or time period.
- **List Reviews** (`lattice_list_reviews`) — List performance review cycles from Lattice. Returns review cycle names, statuses, and date ranges.
- **Get Feedback** (`lattice_get_feedback`) — Get feedback entries from Lattice. Optionally filter by recipient or giver.
