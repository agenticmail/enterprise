# Firebase

Integrate with Firebase to list, get, create and more.

## Tools

- **List Documents** (`firebase_list_documents`) — List documents in a Firestore collection. Returns document IDs and field summaries.
- **Get Document** (`firebase_get_document`) — Get a single Firestore document by its full path (collection/documentId).
- **Create Document** (`firebase_create_document`) — Create a new document in a Firestore collection. Provide fields as key-value pairs.
- **Update Document** (`firebase_update_document`) — Update fields on an existing Firestore document. Only specified fields are updated.
- **Query** (`firebase_query`) — Run a structured query against a Firestore collection. Supports field filters, ordering, and limits.
