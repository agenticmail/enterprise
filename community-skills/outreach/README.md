# Outreach Sales Engagement

Integrate with Outreach Sales Engagement to list, create, list and more.

## Tools

- **List Prospects** (`outreach_list_prospects`) — List prospects from Outreach. Returns prospect names, emails, titles, and companies.
- **Create Prospect** (`outreach_create_prospect`) — Create a new prospect in Outreach. Provide at least an email address.
- **List Sequences** (`outreach_list_sequences`) — List email sequences from Outreach. Returns sequence names, states, and step counts.
- **Add To Sequence** (`outreach_add_to_sequence`) — Add a prospect to an Outreach sequence. Creates a sequence state linking the prospect to the sequence.
- **List Mailings** (`outreach_list_mailings`) — List mailings (sent emails) from Outreach. Returns email subjects, states, and engagement metrics.

| Field | Type | Description |
