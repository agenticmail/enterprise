# Shortcut

Integrate with Shortcut to list, create, search and more.

## Tools

- **List Stories** (`shortcut_list_stories`) — List stories in a Shortcut project. Returns story names, types, states, and assignees.
- **Create Story** (`shortcut_create_story`) — Create a new story in Shortcut. Specify name, project, story type, description, and optional fields.
- **Search Stories** (`shortcut_search_stories`) — Search for stories across Shortcut using a text query. Returns matching stories with IDs, names, and types.
- **List Projects** (`shortcut_list_projects`) — List all projects in Shortcut. Returns project names, IDs, and story counts.
- **List Epics** (`shortcut_list_epics`) — List all epics in Shortcut. Returns epic names, states, and story counts.
