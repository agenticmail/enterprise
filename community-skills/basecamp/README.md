# Basecamp

Integrate with Basecamp to list, list, create and more.

## Tools

- **List Projects** (`basecamp_list_projects`) — List all projects in the Basecamp account. Returns project names, descriptions, and status.
- **List Todos** (`basecamp_list_todos`) — List to-dos in a Basecamp to-do list. Returns to-do titles, assignees, and completion status.
- **Create Todo** (`basecamp_create_todo`) — Create a new to-do in a Basecamp to-do list. Specify content, assignees, and optional due date.
- **List Messages** (`basecamp_list_messages`) — List messages (posts) in a Basecamp project message board. Returns titles, authors, and dates.
- **Create Message** (`basecamp_create_message`) — Create a new message (post) on a Basecamp project message board.
