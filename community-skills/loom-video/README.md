# Loom

Integrate with Loom to list, get, delete and more.

## Tools

- **List Videos** (`loom_list_videos`) — List Loom videos owned by the authenticated user. Returns video names, durations, and share links.
- **Get Video** (`loom_get_video`) — Get detailed information about a specific Loom video including its share link, duration, and view count.
- **Delete Video** (`loom_delete_video`) — Delete a Loom video by its ID. This action is permanent and cannot be undone.
- **Get Transcript** (`loom_get_transcript`) — Get the transcript of a Loom video. Returns the full text transcript with timestamps.

## Configuration

| Field | Type | Description |
|-------|------|-------------|
| Auth | token | loom-video authentication |
