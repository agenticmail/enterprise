# Freshdesk Support

Integrate with Freshdesk Support to list, create, update and more.

## Tools

- **List Tickets** (`freshdesk_list_tickets`) — List support tickets from Freshdesk. Optionally filter by status, priority, or requester email.
- **Create Ticket** (`freshdesk_create_ticket`) — Create a new support ticket in Freshdesk. Provide a subject, description, and requester email.
- **Update Ticket** (`freshdesk_update_ticket`) — Update an existing Freshdesk ticket. Can change status, priority, assignee, or add tags.
- **List Contacts** (`freshdesk_list_contacts`) — List contacts from Freshdesk. Optionally filter by email, phone, or company.
- **Add Reply** (`freshdesk_add_reply`) — Add a reply to an existing Freshdesk ticket. Can send a public reply or an internal note.
