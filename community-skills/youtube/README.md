# YouTube

Integrate with YouTube to search, list, get and more.

## Tools

- **Search** (`youtube_search`) — Search for videos, channels, or playlists on YouTube. Returns titles, channel names, and view counts.
- **List Videos** (`youtube_list_videos`) — Get detailed information about one or more YouTube videos by their IDs. Returns title, description, duration, view count, and engagement metrics.
- **Get Channel** (`youtube_get_channel`) — Get detailed information about a YouTube channel by its ID or custom URL. Returns subscriber count, video count, and description.
- **List Playlists** (`youtube_list_playlists`) — List playlists for a YouTube channel. Returns playlist titles, descriptions, and video counts.
- **Get Analytics** (`youtube_get_analytics`) — Get YouTube Analytics data for the authenticated channel. Returns views, watch time, subscribers gained, and engagement metrics for a date range.
