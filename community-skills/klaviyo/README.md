# Klaviyo

Integrate with Klaviyo to list, create, list and more.

## Tools

- **List Profiles** (`klaviyo_list_profiles`) — List profiles (contacts) from Klaviyo. Supports filtering by email and pagination.
- **Create Event** (`klaviyo_create_event`) — Track a custom event in Klaviyo. Events are used for triggering flows, segmentation, and analytics.
- **List Campaigns** (`klaviyo_list_campaigns`) — List email campaigns from Klaviyo. Returns campaign names, statuses, and send times.
- **List Flows** (`klaviyo_list_flows`) — List automation flows from Klaviyo. Returns flow names, statuses, and trigger types.
- **Get Metrics** (`klaviyo_get_metrics`) — List available metrics (event types) in Klaviyo. Returns metric names, integration sources, and IDs.
