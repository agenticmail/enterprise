# WordPress

Integrate with WordPress to list, create, update and more.

## Tools

- **List Posts** (`wp_list_posts`) — List posts from a WordPress site. Optionally filter by status, category, search, or author.
- **Create Post** (`wp_create_post`) — Create a new post on a WordPress site. Provide a title, content, and optional status and categories.
- **Update Post** (`wp_update_post`) — Update an existing post on a WordPress site. Can change title, content, status, and other fields.
- **List Pages** (`wp_list_pages`) — List pages from a WordPress site. Returns page titles, statuses, and slugs.
- **List Comments** (`wp_list_comments`) — List comments from a WordPress site. Optionally filter by post, status, or author.
