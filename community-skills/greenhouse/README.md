# Greenhouse Recruiting

Integrate with Greenhouse Recruiting to list, get, list and more.

## Tools

- **List Candidates** (`greenhouse_list_candidates`) — List candidates from Greenhouse. Returns candidate names, emails, and application info.
- **Get Candidate** (`greenhouse_get_candidate`) — Get detailed information about a specific Greenhouse candidate by their ID.
- **List Jobs** (`greenhouse_list_jobs`) — List jobs from Greenhouse. Returns job titles, departments, offices, and status.
- **List Applications** (`greenhouse_list_applications`) — List applications from Greenhouse. Returns application status, candidate info, and job associations.
- **Add Note** (`greenhouse_add_note`) — Add a note to a candidate in Greenhouse. Notes are visible to the hiring team on the candidate profile.
