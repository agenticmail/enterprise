# Gong Revenue Intelligence

Integrate with Gong Revenue Intelligence to list, get, get and more.

## Tools

- **List Calls** (`gong_list_calls`) — List calls from Gong. Filter by date range, user, or workspace. Returns call titles, durations, and participants.
- **Get Call** (`gong_get_call`) — Get detailed information about a specific Gong call, including participants, duration, and media.
- **Get Call Transcript** (`gong_get_call_transcript`) — Get the transcript of a specific Gong call. Returns speaker-attributed text with timestamps.
- **List Users** (`gong_list_users`) — List users in your Gong workspace. Returns user names, emails, and statuses.
- **Search Calls** (`gong_search_calls`) — Search Gong calls by keywords, speaker, or content. Returns matching calls with relevance.
