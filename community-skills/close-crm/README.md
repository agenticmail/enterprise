# Close CRM

Integrate with Close CRM to list, create, search and more.

## Tools

- **List Leads** (`close_list_leads`) — List leads from Close CRM with optional pagination.
- **Create Lead** (`close_create_lead`) — Create a new lead in Close CRM. Provide a name and optional contacts, custom fields, and status.
- **Search Leads** (`close_search_leads`) — Search leads in Close CRM using a query string. Supports Close search syntax.
- **Create Activity** (`close_create_activity`) — Create an activity (note, call, or email) on a Close CRM lead.
- **List Opportunities** (`close_list_opportunities`) — List opportunities from Close CRM with optional filtering by lead or status.
