# Supabase

Integrate with Supabase to query, insert, update and more.

## Tools

- **Query** (`supabase_query`) — Query rows from a Supabase table using PostgREST syntax. Supports select, filtering, ordering, and pagination.
- **Insert** (`supabase_insert`) — Insert one or more rows into a Supabase table. Returns the inserted rows.
- **Update** (`supabase_update`) — Update rows in a Supabase table matching a filter. Returns the updated rows.
- **Delete** (`supabase_delete`) — Delete rows from a Supabase table matching a filter. Returns the deleted rows.
- **Rpc** (`supabase_rpc`) — Call a Supabase database function (RPC). Pass the function name and an optional parameter object.
