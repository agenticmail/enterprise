# Freshsales CRM

Integrate with Freshsales CRM to list, create, list and more.

## Tools

- **List Contacts** (`freshsales_list_contacts`) — List contacts from Freshsales CRM with optional filtering and pagination.
- **Create Contact** (`freshsales_create_contact`) — Create a new contact in Freshsales CRM. Provide at least an email or last name.
- **List Deals** (`freshsales_list_deals`) — List deals from Freshsales CRM with optional filtering and pagination.
- **Create Deal** (`freshsales_create_deal`) — Create a new deal in Freshsales CRM. Provide at least a name.
- **Search** (`freshsales_search`) — Search across Freshsales CRM entities (contacts, deals, accounts, etc.) by keyword.

## Configuration

| Field | Type | Description |
|-------|------|-------------|
| Auth | api_key | freshsales authentication |
