# Copper CRM

Integrate with Copper CRM to list, create, search and more.

## Tools

- **List People** (`copper_list_people`) — List people (contacts) from Copper CRM. Copper uses POST for list/search endpoints.
- **Create Person** (`copper_create_person`) — Create a new person (contact) in Copper CRM. Provide at least a name.
- **Search Leads** (`copper_search_leads`) — Search leads in Copper CRM. Copper uses POST for search endpoints.
- **List Opportunities** (`copper_list_opportunities`) — List opportunities from Copper CRM. Copper uses POST for list/search endpoints.
- **Create Task** (`copper_create_task`) — Create a new task in Copper CRM. Tasks can be associated with people, leads, or opportunities.

| Field | Type | Description |
