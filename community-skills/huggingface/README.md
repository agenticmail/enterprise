# HuggingFace

Integrate with HuggingFace to inference, list, get and more.

## Tools

- **Inference** (`hf_inference`) — Run inference on a HuggingFace model. Supports text generation, summarization, classification, and more depending on the model.
- **List Models** (`hf_list_models`) — Search and list models on HuggingFace Hub. Filter by search query, author, or task.
- **Get Model** (`hf_get_model`) — Get detailed information about a specific HuggingFace model including description, tags, downloads, and config.
- **List Datasets** (`hf_list_datasets`) — Search and list datasets on HuggingFace Hub. Filter by search query, author, or task.
- **List Spaces** (`hf_list_spaces`) — Search and list Spaces on HuggingFace Hub. Spaces are hosted ML demo apps (Gradio, Streamlit, etc.).
