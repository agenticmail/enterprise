# ADP Workforce

Integrate with ADP Workforce to list, get, get and more.

## Tools

- **List Workers** (`adp_list_workers`) — List workers from ADP Workforce. Returns a paginated list of all workers with basic information.
- **Get Worker** (`adp_get_worker`) — Get detailed information about a specific ADP worker by their associate OID.
- **Get Payroll** (`adp_get_payroll`) — Get payroll information from ADP. Returns pay statements for a specific worker or a summary of recent payroll runs.
- **List Org Units** (`adp_list_org_units`) — List organizational units from ADP. Returns departments, divisions, cost centers, and other org structures.
- **Get Benefits** (`adp_get_benefits`) — Get benefit enrollment information for a specific ADP worker. Returns active benefits, coverage, and plan details.
