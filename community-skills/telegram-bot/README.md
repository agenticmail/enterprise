# Telegram Bot

Integrate with Telegram Bot to send, send, get and more.

## Tools

- **Send Message** (`telegram_send_message`) — Send a text message via Telegram Bot API. Supports Markdown and HTML formatting.
- **Send Photo** (`telegram_send_photo`) — Send a photo to a Telegram chat. Provide the chat ID and a URL to the photo.
- **Get Updates** (`telegram_get_updates`) — Get incoming updates (messages, callbacks, etc.) for the Telegram bot. Use offset to acknowledge previous updates.
- **Get Chat** (`telegram_get_chat`) — Get information about a Telegram chat (private, group, supergroup, or channel).

## Configuration

| Field | Type | Description |
|-------|------|-------------|
| Auth | api_key | telegram-bot authentication |
