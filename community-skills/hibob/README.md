# HiBob

Integrate with HiBob to list, get, search and more.

## Tools

- **List Employees** (`hibob_list_employees`) — List employees from HiBob. Returns names, emails, departments, and work information.
- **Get Employee** (`hibob_get_employee`) — Get detailed information about a specific HiBob employee by their ID or email.
- **Search Employees** (`hibob_search_employees`) — Search for employees in HiBob using field filters. Supports searching by name, department, title, and custom fields.
- **List Time Off** (`hibob_list_time_off`) — List time-off requests from HiBob. Filter by employee, date range, or status.
- **Get Company Info** (`hibob_get_company_info`) — Get company information from HiBob. Returns company name, sites, departments, and other organizational data.
