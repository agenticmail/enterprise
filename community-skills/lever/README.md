# Lever Recruiting

Integrate with Lever Recruiting to list, create, list and more.

## Tools

- **List Opportunities** (`lever_list_opportunities`) — List opportunities (candidates in pipeline) from Lever. Filter by stage, posting, or archive status.
- **Create Opportunity** (`lever_create_opportunity`) — Create a new opportunity (candidate) in Lever. Provide candidate name, contact info, and optionally associate with a posting.
- **List Postings** (`lever_list_postings`) — List job postings from Lever. Returns posting titles, teams, locations, and states.
- **Get Candidate** (`lever_get_candidate`) — Get detailed information about a candidate (opportunity) in Lever by their opportunity ID.
- **Add Note** (`lever_add_note`) — Add a note to a candidate opportunity in Lever. Notes are visible to the hiring team.
