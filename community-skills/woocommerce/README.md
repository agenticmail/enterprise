# WooCommerce

Integrate with WooCommerce to list, create, list and more.

## Tools

- **List Products** (`woo_list_products`) — List products in the WooCommerce store. Returns product names, prices, stock status, and SKUs.
- **Create Product** (`woo_create_product`) — Create a new product in the WooCommerce store. Provide a name and optional price, description, SKU, and status.
- **List Orders** (`woo_list_orders`) — List orders from the WooCommerce store. Optionally filter by status or date range.
- **Update Order** (`woo_update_order`) — Update an existing WooCommerce order. Can change status, add a note, or update billing/shipping details.
- **List Customers** (`woo_list_customers`) — List customers in the WooCommerce store. Optionally search by name or email.
