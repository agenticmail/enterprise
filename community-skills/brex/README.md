# Brex

Integrate with Brex to list, list, list and more.

## Tools

- **List Transactions** (`brex_list_transactions`) — List Brex card transactions. Optionally filter by date range or user. Returns transaction descriptions, amounts, and dates.
- **List Cards** (`brex_list_cards`) — List Brex cards. Optionally filter by user or card status. Returns card details including last four digits and limits.
- **List Users** (`brex_list_users`) — List users (employees) in Brex. Returns user names, emails, roles, and statuses.
- **Get Account** (`brex_get_account`) — Get Brex account details including the primary cash account balance and status.
- **List Vendors** (`brex_list_vendors`) — List vendors (payment recipients) in Brex. Returns vendor names, payment methods, and IDs.
