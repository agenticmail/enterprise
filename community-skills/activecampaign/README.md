# ActiveCampaign

Integrate with ActiveCampaign to list, create, list and more.

## Tools

- **List Contacts** (`ac_list_contacts`) — List contacts from ActiveCampaign. Optionally filter by email or search query. Returns contact names, emails, and IDs.
- **Create Contact** (`ac_create_contact`) — Create a new contact in ActiveCampaign. Provide at least an email address, plus optional name and phone fields.
- **List Deals** (`ac_list_deals`) — List deals from ActiveCampaign CRM. Optionally filter by stage or search query.
- **List Automations** (`ac_list_automations`) — List automations from ActiveCampaign. Returns automation names, statuses, and trigger info.
- **Create Deal** (`ac_create_deal`) — Create a new deal in ActiveCampaign CRM. Provide a title, value, and pipeline/stage info.
