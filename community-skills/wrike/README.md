# Wrike

Integrate with Wrike to list, create, list and more.

## Tools

- **List Tasks** (`wrike_list_tasks`) — List tasks in Wrike. Optionally filter by folder, status, or custom fields.
- **Create Task** (`wrike_create_task`) — Create a new task in a Wrike folder or project. Specify title, description, assignees, and dates.
- **List Folders** (`wrike_list_folders`) — List folders and projects in Wrike. Optionally filter within a specific parent folder.
- **List Projects** (`wrike_list_projects`) — List all projects in Wrike. Returns project names, statuses, and owner info.
- **Update Task** (`wrike_update_task`) — Update an existing Wrike task. Change title, description, status, assignees, or dates.
